### Instructions

#### Install

First you will need [wkhtmltopdf](http://wkhtmltopdf.org/) (for PDF) and [imagemagick](http://www.imagemagick.org/script/index.php) (for PNG). The
rest can be handled with NPM...

```
npm install
```

#### Build
Edit resume.json and then run this...
```
$ npm run build
```
