### Instructions

#### Install

First you will need [wkhtmltopdf](http://wkhtmltopdf.org/) and [imagemagick](http://www.imagemagick.org/script/index.php).

```
npm install
```

#### Build
Edit resume.json and then run this...
```
$ npm run build
```
