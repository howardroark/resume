### Instructions

> Note: all binaries in the `bin` folder are for OSX. You may need to download your
own at (wkhtmltopdf.org)[http://wkhtmltopdf.org/downloads.html].

#### Install

```
$ npm install
```

#### Build

Edit resume.json and then run this...

```
$ npm run build
```
